# 0xe10c's Library

A place to collect scripts, functions, anything that has proven useful during
work, labs, assessments, exams, etc.

The focus is on reusability. Things in "utility" scripts are mostly just
function definitions for small, repeatable tasks, e.g. base64-encoding a
payload for PowerShell (in pseudocode: `base64_encode(convert_to_utf16le(<payload>))`).

In lieu of redundantly listing the functions and their purposes in this
document, readers are invited to inspect the scripts and function definitions for
themselves; I'm generally pretty good about providing documentation.
